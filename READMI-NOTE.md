## Функции для работы с массивами (основными)

- **Функция для создаия массивов **

``` php 
    array()         -# создает массив
    range()         -# создает массив из диапазона элементов
    array_fill()    -# заполняет массив значениями

```

- **Функция для модификации массивов**

``` php
    array_push()    -# добовляет элементы в конец массива
    array_pop()     -# удаляет последний элемент массива
    array_shift()   -# удаляет первый элемент массива
    array_unshift() -# добовлят элементы в начало массива
    array_splice()  -# удаляет часть массива и заменяет её другой
```

- **Функция для сортировки массива**

``` php
    sort()          -# сортирует массив по возрастанию
    rsort()         -# сортирует массив по убыванию
    asort()         -# сортирует ассоциативный массив по значениям
    ksort()         -# сортирует ассоциативный массив по ключам
    usort()         -# сортирует массив с помощью пользовательской функции
```

- **Функция для поиска в массивах**

```php
in_array()          -# провряет, существует ли значение
array_search()      -# ищет значение в массиве и возвращает ключ
array_key_exists()  -#проверяет, существует ли ключ в массиве
```

- **Функция для работы с несколькими массивах**

```php
array_merge()       -# объединяет два или более массивов
array_diff()        -# вычисляет расхождение массивов
array_intersect()   -# вычисляет пересечение массивов
```



- **Функция для обхода массивов**

```php
array_map()         -# применяет callback-функцию ко всем элементам массива
array_filter()      -# фильтрует элементы массиса с помощью callback-функции
array_reduce()      -#итеративно уменьшает массив к единственному значению
```


- **Другие полезные функции**

```php
count()             -# подсчитывает количество элементов массива
array_keys()        -# возвращает все ключи масива  
array_values()      -# возвращает все значения масива
array_reverse()     -# возвращает масив в обратном порядке
array_unique()      -# удаляет повторяющиеся значения из массива

```



### Циклы: for, while, foreach, их использование для работы с массивами

**Циклы используются для многократного выполнения одного и того же бллока кода**

### **for:**
```php
for($i = 0; $i < count($fruits); $++) {
    echo $fruits[$i]. "\n";
}

```
### **while:**
```php
$i = 0;
while ($i <count($fruits)) {
    echo $fruits[$i]. "\n";
    $i++;
}
```

### **Для Фссоциативных массивов:**
```php
foreach($age an $name => $age_value) {
    echo $name. "is". $age_value. "years old.\n";
}

```

### **foreach (особенно удобен для массивов)**
```php
foreach ($fruits as $fruit) {
    echo $fruit. "\n";
}
```

### **Функции: Объявление, параметры, возрат значений

**Функции --** это блоки кода, которые можно вызвать многократно.

**Объявление функции:**
```php
function functionName($param1, $param2) {
    //code functions
    return $result; //не обязательно
}

```

**Пример/Example**
```php
function add($a, $b) {
    return $a + $b;
}

echo add(2,3);//выведет 5
```

## Работа с формами: методы GET и POST, обработка данных форм, безопасность

**Формы HTML** позволяют пользователю вводить данные, которые затем могут быть обработаны на сервере.

**Форма может использовать методы:**

`GET`- данные передаются в URL(допустимая длина ограничена, видны в строке браузера)

`POST`-данные передаются в теле запроса,(более безопасны для конфедециальных данных, нет ограничений по длине)

## **Пример формы**

```php

<form method="post" action="process.php">
Name: <input type="text" name="name">
Age: <input type="text" name="age"> <input type="submit" value="Submit">
</form>
```

В файле `process.php` можно получить данные следующим методом:
`$name=$_POST['name'];`<br>
`$age=$POST['age'];`<br>

***WARNING***всегда проверяйте и очищайте(sanitize) данные, полученные от пользоателя, чтобы избежать уязвимостей(SQL inject & XSS)
